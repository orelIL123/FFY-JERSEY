כיצד לעדכן את האתר

  1. הכנת תמונה: העלה את תמונת החולצה לשירות אירוח תמונות (כמו Imgur, Cloudinary וכו') והעתק את כתובת התמונה הישירה.
  2. מילוי טופס: מלא את כל השדות בטופס למעלה ולחץ על "צור נתוני חולצה".
  3. העתקת JSON: לחץ על "העתק ללוח" כדי להעתיק את אובייקט ה-JSON שנוצר.
  4. עדכון jerseys.json:
    • עבור למאגר ה-GitHub שלך: https://github.com/orelIL123/FFY-JERSEY
    • נווט לקובץ data/jerseys.json
    • לחץ על כפתור העריכה (אייקון עיפרון)
    • הוסף את אובייקט ה-JSON שהועתק למערך (אל תשכח להוסיף פסיק אם זה לא הפריט הראשון)
    • בצע commit לשינויים
  5. המתן לפריסה: GitHub Pages יעדכן אוטומטית את האתר שלך תוך מספר דקות.
